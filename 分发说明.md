# 分发说明

## 📦 如何分发给其他用户

这个工具**不需要打包**就可以直接分发给用户使用！

---

## 🚀 快速分发方法

### 1. 压缩项目文件夹

将整个 `word-count` 文件夹打包成 zip：
- **Mac**: 右键文件夹 → 压缩
- **Windows**: 右键文件夹 → 发送到 → 压缩文件夹

### 2. 发送给用户

通过任何方式发送压缩包：
- 邮件附件
- 微信/QQ 文件传输
- 网盘分享（百度网盘、OneDrive 等）
- U 盘拷贝

### 3. 用户使用方法

用户收到压缩包后：

**Mac 用户：**
1. 解压缩
2. 双击 `启动程序.command` 文件
3. 自动安装依赖并启动（首次需要1-2分钟）

**Windows 用户：**
1. 解压缩
2. 双击 `启动程序.bat` 文件
3. 自动安装依赖并启动（首次需要1-2分钟）

**Linux 用户：**
1. 解压缩
2. 在终端运行：
   ```bash
   chmod +x 启动程序.command
   ./启动程序.command
   ```

---

## ✅ 这种方式的优势

与打包成 .app/.exe 相比：

| 对比项 | 源代码分发 | 打包应用 |
|--------|-----------|----------|
| **文件大小** | ~100 KB | ~50-100 MB |
| **跨平台** | ✅ 一份代码全平台 | ❌ 每个系统单独打包 |
| **更新** | ✅ 替换文件即可 | ❌ 需要重新打包 |
| **易用性** | ✅ 双击启动脚本 | ✅ 双击应用 |
| **首次启动** | 需要1-2分钟安装 | 立即启动 |
| **安全性** | ✅ 代码可见 | ⚠️ 可能被报毒 |

## 📋 用户使用前提

用户电脑需要满足：

**Mac/Linux:**
- 已安装 Python 3（大多数系统自带）

**Windows:**
- 已安装 Python 3
- 如果没有，启动脚本会提示安装

> 💡 **提示**: 如果用户没有 Python，可以引导他们访问 https://www.python.org/downloads/ 下载安装（5分钟内完成）

## 📝 给用户的使用说明（复制粘贴）

你可以把下面的说明发给用户：

```
【Word 字数统计工具 - 使用说明】

这是一个简单好用的 Word 文档批量字数统计工具。

📌 首次使用步骤：
1. 解压下载的压缩包
2. 双击 "启动程序" 文件（Mac 双击 .command，Windows 双击 .bat）
3. 首次启动需要1-2分钟安装组件，请耐心等待
4. 安装完成后，浏览器会自动打开

📌 如何统计字数：
1. 将 Word 文档（或文件夹）拖拽到网页的上传框
2. 等待几秒，查看统计结果
3. 点击"导出 CSV"可以保存结果

📌 注意事项：
- 仅支持 .docx 格式（Word 2007 及以后版本）
- 统计的是"字符数(不计空格)"
- 使用完毕后，关闭浏览器和终端窗口即可

📌 如果遇到问题：
- Windows 用户：确保已安装 Python 3
- Mac 用户：右键点击启动脚本 → 打开
- 详细说明请查看文件夹内的 README.md
```

## 🎁 推荐分发内容

建议压缩包包含以下文件：

```
word-count/
├── 启动程序.command    # Mac 启动脚本
├── 启动程序.bat        # Windows 启动脚本
├── README.md          # 详细使用教程
├── word_count_web.py  # 主程序
├── templates/         # 网页模板
└── static/            # 样式和脚本
```

**不需要包含：**
- `打包说明.md`（开发者文档）
- `打包Mac应用.sh`（开发者工具）
- `打包Windows应用.bat`（开发者工具）
- `dist/`、`build/`、`*.spec`（打包产物）
- `__pycache__/`（Python 缓存）

## 💡 高级技巧

### 创建快捷方式

**Mac:**
1. 右键 `启动程序.command` → 制作替身
2. 将替身放到桌面或 Dock
3. 重命名为"字数统计"

**Windows:**
1. 右键 `启动程序.bat` → 创建快捷方式
2. 将快捷方式放到桌面
3. 右键快捷方式 → 属性 → 更改图标
