# Word 字数统计工具

一个简单好用的 Word 文档批量字数统计工具。

## 功能特点

- 📊 批量统计多个 Word 文档的字数
- 🎨 界面设计简洁美观
- 📥 支持拖拽上传文件或文件夹
- 💾 可导出 CSV 格式结果

## 下载使用

前往 [Releases](../../releases) 页面下载最新版本：

- **Windows 用户**: 下载 `Word字数统计.exe`
- **Mac 用户**: 下载 `Word字数统计-macOS.zip`

下载后直接双击运行，无需安装任何其他软件。
